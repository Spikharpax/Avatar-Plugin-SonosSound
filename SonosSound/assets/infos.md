# Sonos Sound

Contrôle le volume d'un device Sonos par un bouton dans l'interface d'A.V.A.T.A.R
Nécessite le plugin SonosPlayer

![SonosSound](../core/plugins/SonosSound/assets/images/SonosSound.png)

<BR><BR>
SonosSound est aussi un plugin de démonstration de l'ajout d'éléments avec un menu circulaire dans l'interface d'A.V.A.T.A.R

Pour d'information [Cytoscape Menu](https://github.com/cytoscape/cytoscape.js-cxtmenu)

### Configuration

Ce plugin permet le contrôle d'un device Sonos défini dans la propriété "room" (valeur par défaut: Salon)
